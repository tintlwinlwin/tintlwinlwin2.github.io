Error Guide


TDI Listener Result Error

Result from listener is composed of 3 values, Code, Message and Description. Here we will focus on Code and Message.

Type Code Message Comment
- 0 - Everything working fine, you can see in description a Map to interpret your results after Capturing.
DEP 12 INVALID_PUBLIC_KEY Be sure you called once getProfile API, to get the latest public key.
If still not working, contact us to verify the validity of the public key.
DEP 14 INVALID_TIMESTAMP Check the Date & Time of your phone to be correct.
Preferably use the option Date & Time -> "Use network-provided time"
HTTP 401 Access denied JWT is invalid or expired
HTTP 403 Forbidden Access forbidden
HTTP 404 HTTP not found Scenario name is wrong
HTTP 500 Server not found URL is wrong
HTTP 501 Not Implemented API method not implemented by the server.
HTTP 502 Bad gateway Proxy failed to complete the request. Could be issue on the proxy or in the server
HTTP 503 Service unavailable The service is unavailable
Parsing 1000 Invalid JSON: parsing error Received a non-valid Json
Server 2001 Business critical: [message] Some Check failed eg. Expired Doc, Face, Voice, Specimen...
Server 3000 Internal service error Unexpected error with the platform that needs to be investigated by Thales.
Server 4000 Missing input[: description] Generic problem with scenario due to missing input.
Server 4001 Missing input[: description] Missing input reported by connector.
Server 4300 Invalid input[: description] Generic problem with scenario due to invalid input.
Server 4301 Invalid input[: description] Invalid input reported by connector.
Server 4302 Invalid Input: Document is unreadable Invalid input, document is unreadable
Server 4500 Invalid state[: description] Generic problem with scenario due to invalid state
Server 4501 Invalid state[: description] Invalid state reported by connector
Server 5000 Internal service error Missing or wrong configuration at connector.
Server 5501 External service error Invalid external connectivity, reported by connector.
Server 5502 External service error Invalid external response, reported by connector.
Server 5503 External service error Unauthorized response from external party


Exceptions

Add a try{...}catch(e: TdiThrowableFailure){...} around TDI api - initialize()

Code Message Comment
8000 device is rooted
8001 one or more functions are hooked
8002 device is suspicious
8003 Treat environment has been tampered
10000 JSON Exception: [message] Invalid Profile API format
20000 DEP encodeRequest Exception: [message] E2E Encryption error, please contact us providing the [message]
20001 DEP decodeRequest Exception: [message] E2E Decrypt error, please contact us providing the [message]
- Illegal HMAC! => Multiple request sent in parallel.
20002 X-Http-Response-Status-Code is missing Issue on DEP response which isn't encrypted.
Check the logs for "DEP Status Code: xxx" and see with Server team why X-Http-Response-Status-Code value is missing, provide "Session ID"
Potential reasons:
- Device date/time not synchronised.
- Patch Request too long to be sent. (Max 60sec)
20003 DEP decode response failed. Empty response body.
30000 HTTP exception O.S. can raise an exception in some cases like and the host is not reacheable (the server url is not right set)


Add a try{...}catch(e: Exception){...} around TDI api's - newSession(...), start(...), resume(...), update(...), result(...)

Type Message Comment
IllegalStateException Multiple session creation not allowed. Either stop previous session or continue using previous session Session will be closed only if it status goes to COMPLETED, use TDI stop() call if need to force close the session
IllegalStateException TdiMobileSdk not initialized. Trying to use SDK without initialize it
IllegalStateException TdiMobileSdk session isn't created. SDK is receiving responses from Server or FCM without any Session created. Can happen if force close a Session.
IllegalStateException Task not found. Either session is in completed/pending state or session has not been initialized Trying to Resume non existing task, might have been already executed
IllegalStateException Session cannot be started twice Trying to call start() multiple time
IllegalStateException Cannot invoke action on completed state Trying to execute start(), resume() or update() on a Completed state Session
IllegalStateException Cannot invoke action on pending state Trying to execute start(), resume() or update() on a Pending state Session. Pending state, mean the SDK is waiting for server update before continuing
