Mobile TDI SDK Developer's guide

SDK Version: 2.0.0

Platform: Android

Min OS Version: Android 6 (23)


Overview

TDI SDK is a mobile framework to help to Capture a set of data, described on a Scenario. Once Captured, data will be sent to TDI back to verify the integrity and authenticity of the data to help you to create a Trust Digital Identity.


Purpose

This document provides a description on how to integrate the SDK with an application in order to execute a given Scenario containing a series of Captures to be done. It will explain how to use the provided tools to Capture and execute all the steps of the Scenario to get the Result information about the integrity and authenticity of data previously captured.
